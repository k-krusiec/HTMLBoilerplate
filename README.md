# Boilerplate

Boilerplate for html sites.

## Installation

Requires [Node.js](https://nodejs.org/) v6+ to run.

Install the devDependencies

```sh
$ cd project
$ npm install
```

## Development

Watch + browser sync (in your default browser)
```sh
$ gulp
```

## Production
For production release:
```sh
$ gulp build
```
